# Excel Sync Manager Release

## For end users

1. Open this folder
2. Double-click `Excel Sync Manager.exe`
3. Use the GUI to create tasks
4. Keep `tasks.json` in the same folder as the exe

## Folder notes

- `Excel Sync Manager.exe`: main app
- `tasks.json`: saved tasks and settings
- `_runtime\manager.log`: runtime log

## Important

- Do not move the exe out of this folder by itself
- If you want to move the tool to another PC, copy the whole folder
